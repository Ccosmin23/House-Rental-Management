# House-Rental-Management

### Tools: Intellij 2020 IDEA, SceneBuilder

Project Structure Intellij:
`csv-0.2.1`
`json-simple-1.1`
`string-0.1.0`
`javafx-sdk-16`
`mysql-connector-java-8.0.25`


Check House-Rental-Management `images` folder to see how GUI looks like
